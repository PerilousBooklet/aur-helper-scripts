## AUR

A collection of simple AUR helper scripts.

## Usage

Download a new package with:

`cd ~/.aur/packages`

`git clone url`

Build all downloaded packages with:

`cd ~/.aur`

`./build.sh `

Install all built packages with:

`cd ~/.aur`

`./install.sh`

Update to latest version all downloaded packages with:

`cd ~/.aur`

`./update.sh`
